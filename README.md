# Django_employees

Это Django app, представляющее web сайт, с помощью которого можно управлять 
записями о сотрудниках фирмы. Осуществлять поиск сотрудников в базе данных.
В данной версии, БД содержит 51100 тестовых записей.

  При создании сайта, использованы:
- Django 2.1
- MySQL 5.6
- Python 3.7

  Frontend:
- Bootstrap v4.1
- Jquery + Jquery UI

# Часть 1:

  Создал веб страницу, которая выводит иерархию сотрудников в древовидной форме:
Информация о каждом сотруднике хранится в базе данных и содержит следующие данные:
- ФИО;
- Должность;
- Дата приема на работу;
- Размер заработной платы;

У каждого сотрудника есть 1 начальник.

База данных содержит 51 100 сотрудников и 5 уровней иерархий.

Отображена должность каждого сотрудника.
